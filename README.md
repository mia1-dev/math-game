# math-game
This game randomly generates math questions (addition, subtraction, multiplication, and division) using numbers 1-10. The game continues as long as you answer correctly, and ends when you give a wrong answer.

# How to play
1. The program displays a random math question
2. Enter your answer
3. If your answer is correct, you earn a point and get another question
4. If your answer is wrong, the game ends and shows your final score
